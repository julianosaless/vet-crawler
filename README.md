# vet-crawler


## Installing
* npm run configure 

## Install
* npm run install

## Uninstall
* npm run uninstall


## Running 
* dotnet-vd
  
## Command Line

 * dotnet-vd  --help
 * dotnet-vd 
 * dotnet-vd directory -o "C:\folder"
